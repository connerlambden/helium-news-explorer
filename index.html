🎈 Helium News Bias Explorer

An interactive view of 216 news sources scored across 37 bias dimensions — fetched live from the Helium MCP server. No signup, no key, no tracking. Three modes: rank sources on any dimension, scatter any two dimensions with live Pearson r, or compare two sources head-to-head on a radar chart.

Tries the live API first (heliumtrades.com/mcp_all_source_biases/); falls back to a bundled snapshot if the free quota is exhausted. Source: github.com/connerlambden/helium-news-explorer.
